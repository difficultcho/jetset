# Handoff: JETSET 户外/滑雪购物应用（微信小程序风格）

## Overview
JETSET 是一款瑞士户外/滑雪装备品牌的移动购物应用（微信小程序风格），包含 5 个底部 Tab：首页、商城、购物车、批发、我的。本设计涵盖浏览、下单、购物车管理、批发合作申请、个人信息与地址管理等完整购物闭环。

## About the Design Files
本包中的 `Shopping App.html` 是**设计参考文件**——用 React + 内联 JSX（通过 Babel 单文件运行）搭建的高保真可交互原型，用来展示界面外观与交互行为，**不是可以直接照搬的生产代码**。请在目标代码库现有的技术栈（如 React Native / Taro / uni-app / 原生小程序框架等）中，参照本设计**重新实现**界面与交互；如果项目还没有确定技术框架，请根据项目情况选择最合适的框架来实现。

设计文件用 390×844（iPhone 尺寸）的手机边框包裹并居中显示，仅用于原型演示，实现时应改为正常的小程序/App 页面结构（无需保留黑色手机边框）。

## Fidelity
**高保真（hifi）**：本设计已包含最终配色、字体、间距与交互逻辑，可视为最终视觉稿。开发时应像素级对照本设计还原界面，颜色使用下方"Design Tokens"中列出的精确色值。

商品图片目前是用 CSS 绘制的占位图案（品牌 Logo + 色块），**并非最终商品图**，实现时应替换为真实商品拍摄图或渲染图。

## Interactions & Behavior

### 导航与手势
- **底部 5 个 Tab** 切换时对应内容整体切换渲染；商城/我的内部子页面状态在离开 Tab 时会重置（因为组件被卸载重新挂载）——如果需要保留子页面浏览位置，开发时应考虑用路由或持久化状态代替。
- **商品详情页支持右滑手势返回**列表页：监测 `touchstart`/`touchend`，当水平位移 dx > 80px 且垂直位移 |dy| < 60px 时触发返回。原型中详情页顶部已移除返回箭头（避免遮挡大图），依赖此手势 + 底部 Tab 导航返回。
- 购物车 Tab 图标右上角显示购物车内商品总数量的红色数字徽标（`cnt > 0` 时显示）。

### 表单与校验
- 商品配置选择弹窗：颜色/尺码未选全时，主按钮为禁用态（灰底灰字，`cursor: not-allowed`）。仅当商品只有 1 种尺码/颜色时默认选中，无需用户操作。
- 批发合作申请表单为演示性质，未做实际字段校验，仅在点击"申请合作"后展示成功状态。
- 新增收货地址表单：姓名/手机/地址三个字段全部非空才允许保存。

### 动效
- 首页 Hero 轮播：背景色随幻灯片切换有 0.8s 过渡；文案区块透明度 0.4s 过渡；分页圆点宽度/颜色 0.3s 过渡。
- 各类选中态（颜色块、尺码胶囊、分类项）均为 0.15s–0.2s 的 border/background 过渡。

### 空状态
- 购物车为空、订单列表为空、地址列表为空均有对应插画（当前为简化线框图标）+ 引导文案 + 行动按钮（大多数跳转商城）。

## State Management
建议至少包含以下状态（原型中用简单的 React useState 在根组件 `App` 维护并通过 props 下传）：
- `tab`：当前选中的底部 Tab（'home' | 'shop' | 'cart' | 'wholesale' | 'mine'）
- `cart`：购物车数组，每项包含 `{ prod, ci(颜色索引), si(尺码索引), qty(数量) }`
- `orders`：订单数组，每项包含 `{ id, items, status, date }`
- `addresses`：收货地址数组，每项包含 `{ id, name, phone, address }`
- `userInfo`：个人信息对象 `{ name, phone, gender, birthday, region }`
- 商城 Tab 内部页面栈（列表 → 详情 → 配置弹窗 → 确认订单）
- 我的 Tab 内部页面栈（主页 → 订单/地址/信息）

数据获取：本设计中商品数据（`PRODS` 数组）为前端写死的 mock 数据，实现时应替换为真实 API 请求（商品列表、详情、下单、地址 CRUD、用户信息读写等）。

## Design Tokens

### 颜色
- 主文字色：`#1a1a1a` / `#333`
- 次要文字色：`#666` / `#888` / `#999` / `#aaa` / `#bbb`（由深到浅的灰阶层级）
- 分隔线/描边：`#f0f0f0`（浅）、`#e8e8e8`、`#e0e0e0`、`#ddd`
- 页面背景：`#f5f5f5`；卡片背景：`#fff`
- 主按钮：黑底白字 `#1a1a1a` / `#fff`；次按钮：白底黑边描边
- 强调色（订单状态/标签）：橙色 `#E86A2B` / `#ff4444`（默认标签、缺货提示等）
- "我的"页头部低调奢华配色（v2，替换早期过于鲜艳的金棕色方案）：
  - 背景渐变：`#3a352e → #2b2823 → #211e1a`
  - 金色点缀文字/描边：`#d6be94`，透明度 rgba(214,190,148, 0.18–0.75)
  - 浅米白文字：`#ece6dc`
- 图标线条色（低饱和棕灰）：`#6b6459`
- 商品色块示例（用于图片占位的主题色）：橙 `#E86A2B`、蓝 `#5B9BD4`、黑 `#2a2a2a`、紫 `#B8A6D4`、灰 `#888` 等——对应各商品的可选颜色

### 字体
- 字体族：`'Noto Sans SC', -apple-system, sans-serif`（中文默认字重 400，标题常用 600/700/900）
- 字号层级：9px（英文小标题/辅助说明）、10–13px（正文/标签）、14–16px（正文强调/商品名）、17–28px（价格/大标题）

### 圆角
- 小型标签/按钮：4–6px
- 卡片/图片：6–12px
- 胶囊按钮：20–24px（完全圆角）
- 弹窗顶部圆角：18px

### 间距
- 页面左右内边距：16px（部分紧凑场景 12–14px）
- 卡片内边距：12–16px
- 组件间距（gap）：8–12px 常用

### 阴影
- 手机边框外部阴影（仅原型演示用，非页面内容）：`0 50px 120px rgba(0,0,0,0.85)`
- 页面内容基本不使用投影，以描边/背景色分隔为主（符合微信小程序扁平化风格）

## Assets
- `assets/jetset-logo.png`：JETSET 品牌 Logo（黑色版本，几何山峰造型 + JETSET 字样拼合设计）。深色背景处通过 CSS `filter: invert(1) brightness(1.5–1.6)` 反色显示为浅色版本；浅色背景处直接使用原黑色版本或降低透明度用作水印。
  - 建议开发时向设计要一份官方提供的白色/透明版本文件，避免长期依赖 CSS 滤镜转换颜色。
- 商品图片：原型中全部为 CSS 绘制的占位图案（品牌小字 + 色块圆形），**非最终商品图**，需替换为真实拍摄图。

## Files
- `Shopping App.html` — 完整交互原型（React + Babel 单文件，可直接用浏览器打开预览）
- `assets/jetset-logo.png` — 品牌 Logo 文件

如需查看具体某个界面的精确像素实现（颜色、间距、条件渲染逻辑等），请直接在 `Shopping App.html` 中搜索对应中文文案或组件名（如 `function ProdDetail`、`function CartTab`、`function MineTab` 等），所有界面均以独立 React 函数组件形式组织，便于逐一对照。
